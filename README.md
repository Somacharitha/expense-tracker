# 💰 Smart Expense Tracker (Full Stack)

A full-stack expense tracking web application built using modern technologies.

---

## 🚀 Tech Stack

### 🔹 Backend
- FastAPI
- SQLite
- JWT Authentication
- REST APIs

### 🔹 Frontend
- React
- Tailwind CSS
- Axios

---

## ✨ Features

- User Signup & Login
- JWT Protected Routes
- Add / Edit / Delete Expenses
- Clean UI
- Secure Backend APIs

---

## 🛠 How to Run Locally

### Backend
cd Backend
pip install -r requirements.txt
uvicorn app:app --reload

### Frontend
cd Frontend
npm install
npm start
